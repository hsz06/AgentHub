# AgentHub AI 协作开发记录

本目录记录 AgentHub 开发中的 AI 协作资产，用于对应考察要点中的 “Spec / rules / skill 沉淀”。

## 文档结构

|文件|说明|
|---|---|
|[01-rules-and-prompts.md](01-rules-and-prompts.md)|协作规则、Prompt 模板、Agent 角色模板|
|[02-development-log.md](02-development-log.md)|关键开发迭代、问题诊断和修复记录|
|[03-evaluation-evidence.md](03-evaluation-evidence.md)|评分维度证据、验收命令、答辩材料索引|
|[04-spec-skill-rules-playbook.md](04-spec-skill-rules-playbook.md)|可执行的 Spec / Skill / Rules 协作规范|

根目录 `docs/ai-collaboration-record.md` 是总览版，本目录是详细版。

## 如何使用这些规范

后续继续让 AI 开发 AgentHub 时，推荐顺序是：

1. 先用 `04-spec-skill-rules-playbook.md` 写任务卡。
2. 再按任务类型选择对应 Skill：诊断、前端体验、Runtime 接入、文档补齐或评审。
3. 实现后把问题和验证结果写入 `02-development-log.md`。
4. 答辩时用 `03-evaluation-evidence.md` 对应评分维度。
